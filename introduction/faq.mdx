### Other frequently asked questions

<Accordion title="How do I implement chunking strategies?">
  You can implement chunking strategies by specifying them in the `chunking_strategy` field when you create or update a data source. Current options include language-based, markdown-based, and HTML-based chunking.
</Accordion>

<Accordion title="How do I set up a model chain?">
  To set up a model chain, use the `create_chain` endpoint. You'll need to specify the types of models you want in the chain, the data source, and any chunking strategies you wish to apply.
</Accordion>

<Accordion title="Is there a limit on model chains or data sources?">
  While there's no hard limit on the number of model chains or data sources, performance might vary based on server resources. 
</Accordion>

<Accordion title="Can I use custom models?">
  Custom HuggingFace models will be supported in future releases.
</Accordion>

<Accordion title="How secure is my data?">
  All data is encrypted during transit. We do not store your files on our servers unless explicitly instructed.
</Accordion>

<Accordion title="How do I debug my model chain?">
  Log visibility is available between each model hand-off, aiding in debugging.
</Accordion>
